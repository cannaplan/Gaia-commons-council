# Gaia Commons Council (GCC)

Sustainable school greenhouse network feeding MN K-12 (800k lunches/day), scaling nationally via endowment.

## Mission
End food insecurity: on-site hydroponics + $48B fund (philanthropy/sports partners).

## Quick Start
\`\`\`bash
git clone https://github.com/cannaplan/Gaia-Commons-Council.git
cd Gaia-Commons-Council
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
gaia-council pilot --config config/pilot.json
\`\`\`

## Files
- \`create_gaia_model.py\`: Yield/meals modeling.
- \`gaia_commons_council/\`: Core package.
